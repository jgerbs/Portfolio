### 🛡️ Cybersecurity Algorithm Demonstrations
Browser-based lab for exploring attack mechanics and defensive concepts in a controlled environment.

- Keylogger simulation (consent-gated, browser-only), brute force, dictionary, and heuristic PIN attacks  
- Parallel heuristic search using 4 Web Workers with live thread output  
- Ransomware simulation with staged execution flow and SOC-style log viewer  
- Incident report aligned with MITRE ATT&CK including IOC tracking  

**Stack:** JavaScript, HTML5, CSS3, Web Workers API
